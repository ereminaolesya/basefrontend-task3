https://ereminaolesya.github.io/basefrontend-task3/

## Описание

Реализация HTML+CSS проекта интерактивной коллекции кнопок с различными линейными градиентами и hover-эффектами для демонстрации многообразия градиентных возможностей CSS.
Задача взята из учебного практического курса.

## Требования к задаче

Необходимо реализовать коллекцию из 8-ми кнопок
Каждая кнопка имеет уникальный фон на основе линейных градиентов
При наведении необходимо плавно перемещать паттерн градиента на 100% влево
